# Sketch-Flex

SketchFlex is an application that allows you to take an idea and immediately put pen to paper. This application is designed for those who have the need to quickly sketch out ideas that can be saved for later. The main driving force behind SketchFlex is the need to be able to quickly sketch ideas without all the other unneeded features of other similar applications.
